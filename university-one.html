University Mockup One

Built as a proof of concept and test for use with a custom in-house CMS. The intention was to create a simple frontend structure for a component based site in which those components work as seamless puzzle pieces to be swapped with each other. Use of classes was specific to the CMS for easy customization. The site also needed to pass stringent accessibility checks.
